# Server Utilization Monitor

- A lightweight Node.js script to monitor system performance including CPU usage, memory usage, and system stats in real time. Logs are printed to console and stored in a file for analysis.

🚀 Features
 - 🧠 CPU utilization tracking (per core aggregation)
 - 💾 Memory usage (total, used, free, percentage)
 - ⏱ System uptime
 - 📉 Load average monitoring
 - 📝 Logs saved to file (stats.log)
 - 🔁 Runs automatically at intervals

🛠 Tech Stack
 - Node.js
 - Built-in modules:
   - os
   - fs
